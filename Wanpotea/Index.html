WANPO STORY
萬波島嶼紅茶專門店 Wanpo Tea Shop
『轉進巷子之後，你會看到一個磚紅色大門，門口有一顆針葉樹，那個就是我家』
在那個沒有導航的年代，每當坐計程車回家，我都是這樣跟司機說我在眷村的家。對了，當時的計程車還是暗紅色的。

這個島嶼千千萬萬的故事，都隨著海洋的靜美萬萬不息，浪波迴盪。
在台灣，每個眷村，都有著關於一杯茶的故事。
從小，我就在瀰漫著酸酸甜甜的氣息中長大成人。眷村是一個大社區，鄰居說著各地口音，有著不同的生活習慣和口味，我的童年回憶在這裡穿梭，而來自各地的阿姨鄰居們也都在鳴光商行客串，買了一杯愛玉檸檬，消磨一下午嘮嗑時光。

萬波，想讓這萬波風味傳唱下去。
TASTE / 口感
軟嫩愛玉、好酸檸檬加上特選好茶，完美表現出沁涼又能咀嚼的清爽，有著現代追求的標準化，依舊不失一杯杯眷村滋味。
LEMON / 檸檬
萬波選自台灣在地的檸檬，100%新鮮現榨檸檬汁，外加檸檬片，整杯好檸味。
TEA / 選茶
使用產地直送的台灣茶葉，泡一杯回甘好茶更能提點出檸檬和愛玉的鮮。
  • 01
    QUALITY / 品質
    萬波每一步都堅持從茶園到茶店產地直送。從摘選，焙茶，補火，煮製，釋放難得一見的經典茶香，萬波將40 年代的匠人精神發揮在手搖茶店中。對於萬波來說，遵循古法「煮茶」才能詮釋真正的茶的美學，掌握特性與適當的沖泡手法，並讓精挑細選的配料建築在古早味之中，突顯出不同茶品其獨特的口感與香氣。
  • 02
    SERVICE / 服務
    一個美好的體驗－萬波會從親切的招呼顧客開始，並且專業的為顧客做菜單介紹，透過乾淨明亮的玻璃能夠看到店員製作飲料的過程，拿到飲料後我們會說聲小心拿和路上小心，希望顧客能有在家一樣的氛圍，在這樣繁忙的生活裡頭，能夠得到一點點放鬆的時光。
  • 03
    TEAMWORK / 團隊
    萬波是一個專業且擁有很多想法的團隊，我們致力把生活中的故事分享給消費者－關於台灣眷村。古早味飲品是大多數人的回憶，因此能讓更多人輕易地去連結這段即將逝去的歷史。萬波的團隊互相激盪出新策略、評估可能性、有著因應潮流的彈性以及貫徹的執行力，使團隊成為最有效能的後勤支援，立志做出最台灣味的眷村飲品。
